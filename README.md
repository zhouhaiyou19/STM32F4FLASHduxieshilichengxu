# STM32F4 FLASH读写示例程序

## 简介

本资源文件提供了一个STM32F4系列微控制器的FLASH读写示例程序。该程序展示了如何将数据保存到STM32F4的内部Flash中，并进行读取操作。通过这个示例程序，您可以了解STM32F4的FLASH操作方法，并体验其强大的功能。

## 功能描述

- **FLASH写操作**：将数据写入STM32F4的内部Flash中。
- **FLASH读操作**：从STM32F4的内部Flash中读取已保存的数据。
